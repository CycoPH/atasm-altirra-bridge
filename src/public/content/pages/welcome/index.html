Features

Use ATasm, the 6502 Mac/65 compatible cross-assembler, to build your Atari assembler projects on your PC and run them in the Altirra emulator.

This VSCode extension comes with the latest version of ATasm as found here on GitHub.

You will need to install the Altirra emulator on your machine and setup the path to it.
